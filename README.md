# 使用方法：
说明：
1、image包下放的就是原始需要扫描的图片
2、target_image下为扫描后生成的文件
3、main包下为主函数和功能函数文件

## 仅需将工程（包括image，main，target_iamge三个包）拷贝到任意ide中，目录层级不变，将需要扫描的图片拷贝到image包下，执行主函数即可。其中主函数流程均有注释说明，可选择注释或开启部分功能。
